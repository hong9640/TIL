예시 문제
객체지향 프로그래밍에서 클래스와 인스턴스의 차이를 설명하시오.
답: 클래스는 객체를 생성하기 위한 설계도와 같은 역할을 하며, 객체의 특성과 동작을 정의합니다. 인스턴스는 클래스에서 정의된 설계도를 바탕으로 생성된 실제 객체로, 클래스의 특성과 메서드를 가지며, 여러 인스턴스를 생성할 수 있습니다.
함수와 메서드의 차이점에 대해 설명하시오.
답: 함수는 독립적으로 정의된 코드 블록으로, 특정 작업을 수행하고 결과를 반환하는 기능을 합니다. 메서드는 클래스 안에 정의된 함수로, 해당 클래스의 객체와 연관되어 실행되며 객체의 상태를 변경하거나 객체와 관련된 작업을 수행합니다. 메서드는 객체를 통해 호출됩니다.
함수- 사용하면 재사용성 높아지고 가독성, 유지보수성 향상, 특정 작업을 수행하기 위한 재사용 가능한 코드 묶음, 클래스 외부, 내부 위치
메서드- 객체에 속한 함수, 객체의 상태를 조작하거나 동작을 수행
